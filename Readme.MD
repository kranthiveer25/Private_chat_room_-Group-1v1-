This project is a Secure Peer-to-Peer Encrypted Chat Application designed to provide private, fast, and reliable communication without relying on centralized servers for message storage. It uses modern web technologies and strong cryptographic techniques to ensure that user data remains confidential and protected at all times.
The application enables users to create or join private chat rooms using a shared secret key. This key is used to generate an encryption cipher through the Web Crypto API, ensuring that all messages, files, and media exchanged between users are encrypted end-to-end. Only participants with the correct key can decrypt and read the transmitted data, making unauthorized access practically impossible.
Unlike traditional messaging platforms, this system follows a decentralized peer-to-peer architecture using WebRTC and PeerJS. Messages are transmitted directly between connected devices, reducing dependency on third-party servers and eliminating risks associated with centralized data storage. No chat history, user credentials, or personal information is stored externally, reinforcing user privacy.
The platform supports both private one-to-one conversations and group chats, allowing multiple users to securely communicate in real time. In addition to text messaging, users can share images and files in an encrypted format, ensuring that sensitive content remains protected during transfer. The application also includes features such as member management, connection status monitoring, and automatic peer discovery to enhance usability and reliability.
From a technical perspective, the project demonstrates practical implementation of cryptography, distributed networking, and secure communication protocols in a browser environment. It is built entirely using vanilla JavaScript, HTML5, and CSS3, without relying on heavy frameworks, making it lightweight and efficient.
Overall, this project serves as a strong example of how modern web technologies can be combined to build privacy-focused, secure, and scalable communication systems suitable for educational, research, and personal use.
